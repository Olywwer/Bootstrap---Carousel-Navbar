 Bootstrap Carousel + Navbar

Este projeto é uma página web responsiva desenvolvida com HTML, CSS, JavaScript e Bootstrap.
O objetivo é demonstrar a implementação de componentes modernos de interface, como Navbar responsiva, Carousel automático com efeitos de transição, seção de conteúdo e footer.

A página apresenta um layout simples e organizado voltado para temas de tecnologia, utilizando o framework Bootstrap para garantir responsividade, design moderno e compatibilidade com diferentes dispositivos.

 Funcionalidades

* Navbar responsiva com menu de navegação
* Carousel automático com efeito de transição (fade)
* Seção de conteúdo com cards sobre tecnologia
* Layout responsivo utilizando Bootstrap Grid
* Footer simples com informações do site
* Controle do Carousel usando JavaScript

 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

 Responsividade

O layout foi desenvolvido utilizando Bootstrap, garantindo que a página funcione corretamente em desktop, tablet e dispositivos móveis.
